Product Integrity Bug Burndown

This page shows a burndown chart and list of bugs tracked against a specific Firefox version since a specific date.
  • Hovering over a point in the chart will show the bug counts for that date.
  • Clicking an item in the legend to show/hide the corresponding data.
  • Clicking a bug in the list below the chart will load that bug report in a new tab.
  • Change the cf_tracking_firefox number in the URL to view a different Firefox version.
  • Change the Since date in the URL to view a different timespan (longer timespans will result in longer load times).
Please be patient as data may take several seconds to load. Try reloading the page or using a more recent Since date if nothing has loaded after more than a minute. Send me an email if you continue to have issues.
